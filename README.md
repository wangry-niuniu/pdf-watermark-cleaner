# PDF 水印擦除台

本地网页工具。上传 PDF 后，先显示第一页预览图，再手动框选要擦掉的区域。

适合这两种场景：

- 所有页面同一个位置都有水印，框一次，批量处理全部页面
- 只想处理个别页面，按页码范围单独擦除

## 启动

```bash
cd /Users/niuniu/pdf_watermark_cleaner
python3 app.py
```

浏览器打开 `http://127.0.0.1:5050`。

## 用法

1. 拖动 PDF 到页面左侧上传区
2. 等第一页预览出来
3. 在右侧预览图上拖出一个或多个红框
4. 选择：
   - 套用到全部页面
   - 只处理指定页码，例如 `1,3,8-15`
5. 点击“开始处理”
6. 下载生成的新 PDF

## 原理

- 读取 PDF 第一页生成预览图
- 把你在预览上框出的区域换算回 PDF 坐标
- 在对应页面的对应位置盖白色矩形

不会做 OCR，也不会改动其他没选中的区域。
